# Dashboard Contract

## Governing Principle

```text
What matters should be visible.
What is visible should be actionable.
```

## Core Dashboard Dimensions

- Objective progress
- Agent execution status
- Blockers and risks
- Evaluation results
- Trust status
- Maturity level
- Cost and latency
- Benchmark performance
- Product usage
- Revenue signals

## Final Rule

Human approvals should be informed by live evidence, not guesswork.
